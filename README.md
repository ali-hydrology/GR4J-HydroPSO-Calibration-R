# GR4J Hydrological Model Calibration using HydroPSO in R

This repository presents a workflow for calibrating the GR4J rainfall–runoff model using Particle Swarm Optimization implemented in the hydroPSO package in R.

The approach integrates hydrological modeling with automated parameter optimization, allowing users to efficiently calibrate GR4J and evaluate model performance for watershed hydrology studies.

## Model Overview
The GR4J model is a conceptual rainfall–runoff model widely used in hydrology for simulating daily streamflow based on precipitation and potential evapotranspiration inputs.

This workflow combines:

- GR4J rainfall–runoff simulation  
- Automatic parameter calibration  
- Optimization using Particle Swarm Optimization  
- Hydrological model performance evaluation  
- Diagnostic plots for model assessment  

## Why use Particle Swarm Optimization?

Particle Swarm Optimization provides a robust method for exploring the parameter space of hydrological models.

Compared with manual calibration, PSO offers:

- Automated parameter search  
- Improved calibration efficiency  
- Better exploration of the parameter space  
- Reproducible calibration workflows  

This makes it particularly suitable for hydrological model calibration and watershed studies.
